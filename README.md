## 📋 Installation & Deployment

### Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/kodiprabhu29-pixel/kodimarketsfx.git
   cd kodimarketsfx
   ```

2. **Open in browser**
   - Double-click `index.html`
   - Or use Live Server in VS Code
   - Or use Python: `python -m http.server 8000`

### GitHub Pages Deployment

1. **Create GitHub Repository**
   - Repository name: `kodimarketsfx`
   - Make it public

2. **Push Files**
   ```bash
   git add .
   git commit -m "Initial commit: KODI MARKETS FX website"
   git push origin main
   ```

3. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose `main` branch and `/(root)` folder
   - Click Save

4. **Your site is live!**
   - URL: `https://username.github.io/kodimarketsfx/`

## 🎨 Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #000000;
    --secondary-color: #d4af37;
    --accent-color: #1a1a1a;
    --text-color: #ffffff;
    /* ... more variables ... */
}
```

### Market Data
Update market symbols and prices in `index.html`:
```html
<div class="ticker-item">
    <span class="ticker-symbol">EUR/USD</span>
    <span class="ticker-price">1.0875</span>
    <span class="ticker-change positive">+0.25%</span>
</div>
```

### Social Links
Update Instagram handle in links:
```html
href="https://instagram.com/kodi_markets_fx"
```
